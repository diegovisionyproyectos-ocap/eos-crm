# CRM VENTAS APP EOS

Frontend estático (HTML/CSS/JS) para **CRM de ventas EOS** con:

- Leads (pipeline básico)
- Vendedores
- Visitas (check-in) con mapa
- Conexión a Supabase (DB) para guardar y sincronizar

## Archivos

- `eos-crm.html` (entrada principal)
- `eos-crm.css`
- `eos-crm.js`
- `eos-supabase.sql` (tablas/policies para Supabase)

## Cómo abrir en local

Abre `eos-crm.html` con doble clic o con un servidor local.
